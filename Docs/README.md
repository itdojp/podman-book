# Book Publishing Template v3.0 - デザインプレビュー

このページでは、Book Publishing Template v3.0で提案されているモダンデザインシステムのプレビューを確認できます。

## 🔗 プレビューページ

**[📱 デザインプレビューを見る](https://itdojp.github.io/book-publishing-template2/)**

## 🎨 確認できる要素

### 1. レイアウト構造
- サイドバーナビゲーション（進捗表示付き）
- 固定ヘッダー（検索・テーマ切り替え）
- レスポンシブデザイン

### 2. タイポグラフィシステム
- 階層的なフォントサイズ（36px → 14px）
- 最適な行間・文字間隔
- 読みやすさ重視の設計

### 3. コードブロック
- シンタックスハイライト
- コピーボタン
- 言語ラベル表示

### 4. ナビゲーション
- パンくずリスト
- ページ間移動（前へ/次へ）
- 目次への戻るボタン

### 5. カラーシステム
- ライト/ダークテーマ
- プライマリカラー
- セマンティックカラー（成功・警告・エラー）

### 6. レスポンシブデザイン
- デスクトップ（1024px+）
- タブレット（768px-1023px）
- モバイル（767px以下）

## 🌙 テーマ切り替え

プレビューページの右上にあるボタン（🌙/☀️）をクリックすると、ライトモードとダークモードを切り替えできます。

## 📱 モバイル対応

スマートフォンやタブレットでアクセスすると、モバイル最適化されたレイアウトを確認できます。

## 🔄 更新情報

このプレビューは[GitHub Issue #1](https://github.com/itdojp/book-publishing-template2/issues/1)の提案内容を視覚化したものです。
